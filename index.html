PROJECTS

Apostle Engine

The Apostle Engine is an in-development 2D/3D game engine. The engine currently supports Windows and is exclusively using OpenGL as it's graphics API, with plans to eventually transition to using Vulkan and Direct3D.

Apostle is desinged to be a lightweight, multi-platform tool compared to industry standard applications such as Unity and Unreal, and is designed for small teams to be able to work with efficiently.
